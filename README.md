# Curriculum Vitae (CV)

This repository contains my personal Curriculum Vitae, designed with a clean, modern, and professional aesthetic using HTML and CSS.

## 📄 Overview
This CV is built to be lightweight, fast-loading, and easily readable. It utilizes a mobile-responsive design, ensuring it looks great on both desktops and mobile devices.

## 🛠 Technologies Used
* **HTML5:** Structured semantic markup.
* **CSS3:** Styling, layout, and responsive design.
* **Google Fonts:** Using *Roboto* for a clean, professional typography.

## 🚀 Features
* **Responsive Layout:** Automatically adjusts for different screen sizes.
* **Google-Style Aesthetic:** Clean whitespace, structured typography, and modern color palette.
* **Print Friendly:** Designed to look great if saved as a PDF.

## 📋 How to View
You can view the live version of this CV by simply opening the `index.html` file in any modern web browser.
